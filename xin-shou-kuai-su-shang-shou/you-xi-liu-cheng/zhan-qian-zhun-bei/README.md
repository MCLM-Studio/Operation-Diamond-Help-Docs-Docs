# 战前准备

从[系统商店](xi-tong-shang-dian.md)或[玩家市场](wan-jia-shi-chang.md)内购买或者[从仓库中取出](si-ren-cang-ku.md)各种所需要的[战斗用品](/broken/pages/ypW0FfGHrfxtHfLIFoww)

通常来说，你应该携带

* 适量的[武器](/broken/pages/21BxWUsE7BT6MOXXeOQS)（远程武器、近战武器、首领信物）
* 适量的[防具](/broken/pages/AxpjOEIVLNpITeOGLxG2)（头盔、胸甲、护腿、靴子）
* 适量的且有需要用到的[消耗品](/broken/pages/hm7FMBkfn1ZZHLaejg9X)（食物、医疗用品、弹药、投掷物、药水）

如果你想要赚取更多的钱财，也可以携带[钥匙](/broken/pages/xVirf40qIWD46UY6fKsD)去开启[钥匙房](../../tie-men-zen-me-da-kai.md)房间。而且[钥匙](/broken/pages/xVirf40qIWD46UY6fKsD)是唯一的一种不会在你死亡时掉落的物品，不管是局外带入的还是局内获得的都会受到保护，但你可以选择主动丢掉它

<mark style="color:red;">请不要将你的仓库兑换券或者变卖物带入局内，这会在你死亡时掉落</mark>

检查好你的携带物品后你就可以[进行下一步](../zu-dui-xi-tong.md)了
